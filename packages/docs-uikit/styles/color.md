# Colors

默认情况下，控件色调以灰阶色为主，其他颜色作为对应项目的辅助色，每个配色分为 亮（Light）/ 暗（Dark）两套用色规范。分别用在界面设计中的 明亮主题（Light Mode）/ 黑暗主题（Dark Mode）。

## Greyscale 灰阶色

灰阶色见于文本色，图标颜色。左列为明亮主题下的配色，右列为黑暗主题下的配色。
